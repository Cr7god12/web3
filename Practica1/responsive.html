Layout Responsivo

DEFINICION

El diseño responsivo combina técnicas como media queries para adaptar el diseño según el tamaño de la pantalla. Incluye límites máximos y mínimos para mantener la legibilidad y la estructura del contenido. El contenedor responsivo en tu código tiene un max-width de 1200px y un width del 100%. Esto significa que se expandirá hasta 1200px y se reducirá si la pantalla es más pequeña.

Este contenedor se adapta al ancho de la pantalla con un máximo de 1200px.
