Layout con Flexbox

DEFINICION

Flexbox es un sistema de diseño unidimensional que organiza los elementos en una fila o columna y permite distribuirlos de manera flexible. En tu código, se usa justify-content: space-around para distribuir los elementos de manera equitativa. Flexbox es ideal para componentes alineados horizontal o verticalmente.
