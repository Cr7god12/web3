Layout con CSS Grid

DEFINICION

CSS Grid permite organizar el contenido en una cuadrícula bidimensional, dividiendo el espacio en filas y columnas. En tu ejemplo, la cuadrícula está configurada con grid-template-columns: 1fr 2fr 1fr, lo que significa que la segunda columna es el doble de ancha que las otras dos. Grid es muy flexible y adecuado para diseños complejos.
