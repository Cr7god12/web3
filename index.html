Tarea de Web: 3 Tipos de Layouts

¿Qué es un Layout?

La palabra layout sirve para hacer referencia a la manera en que están distribuidos los elementos y las formas dentro de un diseño. Se traduce al español como "disposición", "plan" o "diseño".

Es la representación de un plano sobre el cual se distribuye un espacio específico. En desarrollo web, el layout es la base sobre la cual se construye una página o sitio web.
